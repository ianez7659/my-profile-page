# Brief Introduction of MDX

All blog posts are made with **MDX**. I want to intoroduce some types of callout components we can use in this blog:

<Callout type="tip">
  ### Type: tip

This is my first callout component with tip. The style is controlled by MDX style commands like "#" and "##".

</Callout>

<Callout type="info">
  ### Type: info

This is my 2nd Callout component with info type! This is an example that demonstrates how you can structure paragraphs in a Callout using not only MDX styling, but also by directly using JSX and CSS.

![Sample](/images/me.jpg)

Also, as you can see the image, you can even bring images inside of callout component.

</Callout>

<Callout type="warn">
  ### Type: warn

This is my 3rd Callout component using warn! As you can see, all the types are distinguished by color, so you can freely use the callout type that fits your purpose.

</Callout>
